Orden de arranque (no es un servidor: esto es solo un acceso directo)
  1. Arranca Vite en el proyecto dm_editor: icono del escritorio Iniciar DM Editor, o npm run dev (deja la terminal abierta).
  2. Mira en esa terminal la línea Local: http://localhost:… — si no es el puerto de abajo, abre este archivo con ?base= esa URL.
  3. Luego usa los botones verdes o Comprobar dev server.
Sin comprobar — si aún no arrancaste Vite, es normal. Usa el recuadro verde de arriba.
